Lisez moi du script Pds-Contact-Simple V1.0

Présentation et Prérequis

Pds-Contact-Simple est un script dévéloppé en utilisant PHP, JQuery, HTML, Ajax et Javascript. Il vous permettra d'installer facilement sur votre site un formulaire de contact qui permettra à vos visiteurs de vous contacter par email.

D'utilisation très souple et facile, il vous suffit d'inclure une fonction dans une de vos pages PHP afin d'afficher le formulaire.

Le script est très facilement adaptable au graphisme de votre site via le fichier CSS.

Ce script nécessite seulement que PHP version 5 soit installé sur votre serveur.

L'utilisation de Jquery et Ajax permet l'envoi du mail sans rechargement de la page.

Installation

Pour installer le script, il vous suffit de suivre les étapes suivantes:
  • 1/ Ouvrez le fichier Config.php et indiquez vos informations
  • 2/ Transférez tous les fichiers sur votre serveur, soit à la racine du site, soit dans un repertoire
  • 3/ Sur votre page PHP, celle où vous souhaitez que le formulaire s'affiche, et avant tout code HTML , insérez le code suivant (Adaptez le code selon le répertoire du script):
<? php
require_once("ConfigPds.php");
require_once("Fonctions.php");
CreerFormulaireContact();
?>
  • 4/ Ensuite, toujours sur la page où et à l'endroit où vous souhaitez afficher le formulaire, insérez le code suivant:
<? php
CreerFormulaireContact();
?>
  • 5/ Transférez la page que vous venez de modifier vers votre serveur, lancez ensuite cette page et testez le résultat;) C'est tout!
Nota :Si le fichier où vous souhaitez afficher le formulaire n'est pas dans le même répertoire que le script, vous devez déplacer le fichier Monofont.TTF dans le répertoire où se trouve le fichier qui affichera le formulaire!
